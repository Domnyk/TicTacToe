## TicTacToe

### Description
Tic tac toe game on 5x5 board. Written in Java using JavaFX for GUI and minimax algorithm for computer moves

  
### How to build

#### Windows
    gradlew.bat jfxJar
    java -jar  ./build/app/TicTacToe.jar

#### Linux & MacOS
    ./gradlew jfxJar
    java -jar  ./build/app/TicTacToe.jar
